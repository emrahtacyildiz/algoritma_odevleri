# Merge Sort Projesi

https://app.patika.dev/courses/veri-yapilari-ve-algoritmalar/merge-sort-proje

**[16,21,11,8,12,22]** dizisinin merge sort aşamaları

                    [16,21,11,8,12,22]
            [16,21,11]               [8,12,22]
        [16,21]     [11]        [8,12]       [22]
    [16]    [21]    [11]      [8]   [12]     [22]
         [16,21] [11]             [8,12]  [22]
            [11,16,21]              [8,12,22]
                    [8,11,12,16,21,22]

    => [8,11,12,16,21,22]

Big-O Notation

    2^x = n
    logn = x
    O(logn) O(n)
    => O(nlogn)
